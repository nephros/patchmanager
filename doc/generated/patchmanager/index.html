Applications

The Patchmanager project includes the following applications:

GUI Applications

  • Patchmanager Settings Plugin, a plugin for the Jolla Settings to launch the application
  • Patchmanager QML Plugin, the main UI Application
  • Patchmanager Startup Dialog, a UI shown at Lipstick startup.

Infrastructure Applications

  • Patchmanager Daemon
  • Preload Library
  • Web Catalog
