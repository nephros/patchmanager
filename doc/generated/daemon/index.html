Patchmanager Documentation: Daemon

Overview

A D-Bus activated background service which manages patch un/installation, listing, de/actvation, and communication with the preload library.

Patchmanager is usually launched by its DBus service.

If the patchmanager binary is called from command line, it can also serve as a simple CLI to a running daemon. See the output of patchmanager --help and PatchManagerObject::process() for more information.
