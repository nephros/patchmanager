Patchmanager Documentation: QML Plugin

Overview

The QML Plugin makes up the main Patchmanager GUI Application.

Jolla Settings loads PatchManagerPage from the QML Plugin.

It is a both a front-end to the Daemon and a Patch management application.

It allows to:

  • View the list of installed and activated Patches.
  • Configure the Application and Daemon settings.
  • Activate or deactivate Patches
  • Install Patches from the Web Catalog
  • Uninstall installed Patches

QML types:

PatchObject

An element to be used as content of an PatchManagerModel

PatchManagerModel

A ListModel containing the metadata of Patches

PatchManager

Singleton providing access to the PatchManager methods and properties

PatchManagerTranslator

This Singleton allows patches to include localizations in their shipped QML files

WebPatchesModel

Holds elements from the Web Catalog

WebDownloader

Downloads Patch archives from the Web Catalog
