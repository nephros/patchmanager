QML Components

AboutPage

Shows brief information about the app, with links to Credits, Attributions etc

DevelopersPage

Shows detailed information about the present and past developers of the App

ItemErrorComponent

Shows an error message

PatchManagerPage

Main Patchmanager GUI

RestartServicesDialog

Service restart confirmation dialog

ScreenshotsPage

Shows Screenshots available for a Patch from Web Catalog in a full-screen view

SettingsPage

Manages Application Settings, both global and user-specific ones

UnifiedPatchPage

Shows details about an installed Patch

WebCatalogPage

Shows the list of Patches available in the Web Catalog

WebPatchPage

Shows details about a Patch from the Web Catalog
