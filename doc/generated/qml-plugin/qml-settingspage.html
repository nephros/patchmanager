SettingsPage QML Type

Manages Application Settings, both global and user-specific ones More...

Import Statement: import
  • List of all members, including inherited members

Detailed Description

The usual, system-wide configuration values are set via D-Bus plugin by the Patchmanager daemon, which stores them in /etc/patchmanager2.conf The configuration group uisettings is for settings which solely affect the PM GUI application and consequently also are per-user settings.
