Detailed Description

See also Web Catalog and Patch JSON metadata file.

Property Documentation

fetching : bool

Indicates whether data is currently being downloaded. The WebPatchPage will show a ViewPlaceholder if true.

Default true


isInstalled : bool

true if a version of the patch is currently installed


modelData : var

This property holds the metadata from the model

See also PatchManagerModel and PatchObject.


patchData : var

This property holds the metadata downloaded from the Web Catalog

See also modelData.


versions : var

This property holds a map of patch [name, [versions]]


voteAction : int

State of the like/dislike voting action

MeaningValue
Like2
Dislike1
Not Voted0

See also PatchManager::checkVote.
