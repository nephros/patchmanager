RestartServicesDialog QML Type

Service restart confirmation dialog More...

Import Statement: import
  • List of all members, including inherited members

Detailed Description

This Dialog is shown to the user to confirm restarting of "Services", i.e. anything that has been affected by activated patches.

If accepted, these programs will be killed.

The services to kill are selected via the category field of Patch metadata.
